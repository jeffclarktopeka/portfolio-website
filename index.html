HTML

Hyper Text Markup Language or HTML is the language to "markup" a web page. HTML Tags are used to display the different types of information on a webpage. It is important to note that all the HTML does is create the bare framework of the webpage and DOES NOT create any style on the page, although some HTML can have default styling.

Learn more

CSS

Cascading Style Sheets or CSS, is used to style a webpage in conjuntion with the HTML markup. When using CSS, there are three distinct selector types used to call a particular element of HTML. Those three type types of selectors are element name, "class", or "id". It is always important to make sure your HTML and CSS pages are linked so that all changes you make coincide with each other.

Learn more

Javascript

JavaScript or JS, is a programming languege used to update and change both HTML and CSS. JavaScript uses multiple types of variables to either calculate, manipulate and/or validate data.
